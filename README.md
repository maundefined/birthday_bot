# 🎂 Birthday Bot - Telegram Gift Exchange Manager

A sophisticated Telegram bot built with aiogram 3.x for managing birthday celebrations and gift exchanges within communities. The bot automates the entire process from birthday reminders to gift tracking.

## ✨ Features

### 🎁 For Users
- **Profile Management**: Register with name, birthday, and participation type (give only / give & receive)
- **Wishlist & Addresses**: Specify gift preferences and delivery addresses (OZON, Yandex, Wildberries)
- **Birthday Reminders**: Get notified 10 days before birthdays to update addresses
- **Gift Coordination**: Receive notifications about upcoming birthdays with delivery addresses
- **Barcode Tracking**: Send and receive gift barcodes with sender identification
- **Delay Options**: Postpone gift sending if unavailable

### 👨‍💼 For Admins
- **Full User Management**: View, edit, and delete user profiles
- **Birthday Calendar**: See all birthdays sorted by month, alphabetically, or by registration date
- **Notification Tracking**: Monitor sent notifications and their status
- **Statistics Dashboard**: View bot usage metrics and participation analytics
- **Export Functionality**: Export birthday data for external use

## 🛠 Tech Stack
- **Python 3.9+**
- **aiogram 3.x** - Asynchronous Telegram Bot framework
- **SQLite** - Lightweight database for user data
- **Python-dotenv** - Environment configuration
